# AI-Code-Reviewer
The AI Code Reviewer is a Flask-based web application that analyzes source code and provides coding improvement suggestions automatically.

# AI Code Reviewer

## Project Overview

The AI Code Reviewer analyzes source code and provides improvement suggestions automatically.

The project demonstrates static code analysis and Flask web development concepts.


# Features

- Source code analysis
- Coding suggestions
- Web interface
- Static analysis checks


# Technologies Used

- Python
- Flask
- Regex


# Installation

Install Flask:

pip install flask


# Run Project

python app.py


# How It Works

1. Paste source code
2. Analyze patterns
3. Detect issues
4. Display suggestions
